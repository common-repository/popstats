Popstats is a plugin to enhace statics of your blog, now you'll know more about your visitors.

== Description ==
Popstats is a powerfull realtime stats tool for your wordpress, it makes you get to know more about your visitors.

With PopStats now you can know:
* where are your visitors coming from?
* How many users are on line.
* What browser are they using?
* What OS are they using?
* What's the max number of users conected at same time.
* If a user is on line, you can see what is he/she watching.

Popstats is a wordpress Plugin who was forgoten in the past, but i recover it and make it better. The project was started by Luis Sancho with the first version, 
and sometime before Andres Nieto made the second version, but now popstats is enhaced and updated by me, Victor Martinez(Oxigen).
 
This version, can recognize all the new navigators like Chrome of Safari for Windows, and that's why i updated the code, because before safari just ran over MacOS
but now is available a version for Windows. And the Style was updated to look like the gray dashboard of wordpress.


== Installation ==


1. Upload `wp-popstats.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can find the Popstat's link on the main dashboard menu.

== Upgrade Notice ==

The popstats 2.0 is fully compatible with this version, you can upgrade without fear.
Now Popstats 3.1 has some mobile suport for count Iphone, Ipad, Symbian, Android, PalmOS and other phones.

== Frequently Asked Questions ==

= Does popstats need an acount in the global dashboard of wordpress? =

No, it doesn't. Popstats creates some tables on your database and makes it's own statistics.
